# About
Nextcloud tor client.

# Install
**Debian 12**
```
sudo apt-get install libcurl4-openssl-dev
```

```
git clone https://github.com/oldteamhost/Bridge.git
cd bridge
make
```

**Arch 2023**
```
sudo pacman -S curl
```

```
git clone https://github.com/oldteamhost/Bridge.git
cd bridge
make
```
